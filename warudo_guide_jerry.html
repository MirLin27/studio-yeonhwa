제리님 Warudo 세팅 가이드

순서대로 따라해주시면 돼요! 막히시면 편하게 문의해 주세요 💗

📁 STEP 1 · JSON 파일 넣기
먼저 보내드린 Character_Jerry.json 파일을 Warudo에 임포트해야 해요!
1
좌측 하단 ··· (점 3개) 버튼 클릭
화면 왼쪽 아래에 있는 점 3개 버튼을 눌러주세요.
2
Import asset from file 클릭
메뉴가 열리면 Import asset from file을 선택해주세요.
3
Character_Jerry.json 파일 선택
보내드린 Character_Jerry.json 파일을 찾아서 열어주세요.
파일을 선택하면 좌측 패널에 Character_Jerry가 추가됩니다!
✅ JSON 파일을 먼저 넣어야 다음 단계가 가능해요. 순서를 꼭 지켜주세요!
🧍 STEP 2 · 캐릭터 파일 적용
이제 보내드린 .warudo 파일을 캐릭터 폴더에 넣고 연결해줘야 해요!
1
우측 패널에서 Open Characters Folder 클릭
Character_Jerry를 선택한 상태에서 우측 패널 아래쪽
Open Characters Folder 버튼을 클릭해주세요.
2
열린 폴더에 .warudo 파일 넣기
폴더가 열리면 보내드린 .warudo 파일을 그 폴더 안에 복사해 넣어주세요.
📂 폴더가 열리지 않으면 Warudo를 한 번 껐다가 켜본 후 다시 시도해보세요!
3
Source 드롭다운에서 파일 선택
우측 패널의 Source 항목 드롭다운을 클릭하면
방금 넣은 파일 이름이 보여요. 선택하면 캐릭터가 적용됩니다!
✅ Source에서 파일을 선택하면 제리님 아바타가 화면에 나타납니다!
😊 STEP 3 · 표정 단축키 생성
넣어드린 표정들을 키보드로 실행할 수 있게 단축키를 생성하는 방법이에요!
1
Character_Jerry 선택 후 Expressions 탭으로 이동
좌측에서 Character_Jerry를 선택하고, 우측 패널에서 Expressions 탭을 클릭해주세요.
아래처럼 표정 목록이 보이면 정상이에요!
2
Generate Key Binding Blueprint 클릭
스크롤을 내려서 Generate Key Binding Blueprint 버튼을 클릭하면
아래와 같은 창이 떠요! 단축키가 자동으로 생성된 거예요 🎉
🎹 생성된 표정 단축키
단축키표정
Alt + 1⭐ 표정_별눈
Alt + 2🌸 표정_홍조
Alt + 3😊 표정_^
Alt + 4😠 표정_화남
Alt + 5💧 표정_눈물 1
Alt + 6💧 표정_눈물 2
Alt + Backspace표정 초기화
✅ 이 과정까지 완료되면 Warudo 세팅이 모두 끝납니다 🎉
⚠️ 참고 사항
순서는 반드시 JSON 먼저 → 캐릭터 파일 → 소스 선택 순으로 진행해주세요!
파일이 보이지 않거나 오류가 나면 Warudo를 재시작한 후 다시 시도해보세요.
진행 중 막히시는 부분이 있으면 언제든 편하게 말씀해 주세요 🌸
